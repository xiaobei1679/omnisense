# 知微 (ZhiWei) · 多模态感知系统原型（零 API Key 版）

零依赖 Node ESM 脚手架，演示「看 → 听 → 思 → 说 → 学」闭环。
**全程不调用任何云端 API / 不依赖任何 API Key**——视觉、听觉用真实本地解码，思考用本地符号推理，学习靠克隆公开开源项目。

## 运行

```bash
cd prototype
node src/index.mjs
```

预期输出：真读一本小说（视觉）+ 本地解码一张图 + 听一期转录（听觉）+ 本地分析一段音 → 脑做因果/类比/矛盾/溯因推理 → 嘴主动表达观点并追问 → 用户回应触发重推理 → 心跳克隆开源项目学到「怎么真正去看/听/思/说」的技法。

## 四模块 + 感知融合（全部本地、无 API）

| 模块 | 真实本地能力 | 诚实边界 |
|------|-------------|---------|
| 眼（视觉） | 读文本抽取实体/关系/主题；**本地 PNG 像素解码**（尺寸/主色/亮度/边缘密度/3×3 空间亮度网格/颜色丰富度，Node zlib）；**文档感知**（标题/列表/关键句抽取） | 不做物体识别（需 VLM，但禁用 API）；视觉描述为像素统计启发式 |
| 耳（听觉） | 听转录抽取结构；**本地 WAV 特征**（RMS 能量/过零率/**基频(自相关)**/**谱质心(DFT)**/**静音段占比**，无 ASR） | 不做语音转写（需 ASR，但禁用 API）；基频/谱质心对纯音准、复音为近似 |
| 脑（思考） | **本地符号推理**：因果链、类比、矛盾检测、**归纳**(多主体共享关系→规律)、**演绎**(因果前提+中介→间接结论)、**信念修正**(矛盾时下调置信度并标记存疑)、**意图推断**(用户回应分类)、信念管理（置信度+反方） | 是可解释符号推理，非神经网络/意识 |
| 嘴（交互） | 主动表达观点（带反方、措辞轮换）、基于缺口追问、**追问延伸**(承上启下)、**teach-back**(复述所学)、**满足信号检测**(停问)、回合对话 | 规则/NLG 生成，非自由对话 LLM |
| 感知（融合） | **感知融合层** `perception.mjs`：汇聚眼/耳 percept，构建统一情境模型，按模态覆盖给实体排注意力优先级，产出可解释关注建议 | 仅聚合统计，不引入新推理 |

## 学习 & 心跳

- `learner.mjs`：克隆公开 GitHub 项目（无 API），读取 README/源码注释，蒸馏「如何真正去看/听/思/说」的技法；网络不可达时自动回退到本地已策展的项目目录（如已安装的 `cangjie-skill`、本地框架仓库）继续真蒸馏。
- `heartbeat.mjs`：每小时周期。扫描知识缺口 → 找项目学习 → 把技法纳入记忆库。
- `heartbeat-cron.mjs`：供调度器（WorkBuddy 每小时自动化）调用的**单周期入口**，跑一次即退出。
- 可注册为 WorkBuddy 每小时自动化（`FREQ=HOURLY`；**自动化 id 为机器相关，每台机器不同**，请在本机自动化列表中查看，不要硬编码具体 id）。`heartbeat-cron.mjs` 保留 `memory.json`，各次运行之间持续累积学习与记忆（已学技法按仓库去重，不无限膨胀）。**诚实说明**："持续成长"的边界取决于学习源清单——默认来自代码内策展仓库；往同目录 `sources.json` 追加 `{topic,repo,mapsTo}` 即可扩展新源，否则跑满策展源后每周期仅"巡检"不再新增（绝不假装学到新东西）。手动触发：
  ```bash
  cd prototype && node src/heartbeat-cron.mjs
  ```

## 设计要点

- **解耦**：模块只通过事件契约（`percept` / `insight` / `gap` / `learned` / `user-percept` / `utterance`）通信。
- **零依赖**：仅用 Node 内置模块（`node:fs` / `node:zlib` / `node:child_process`）。
- **诚实降级**：任何本地解码失败都打印降级说明，绝不假装成功；网络不可达时心跳优雅跳过。
- **可解释**：所有「思考」产物都带证据与反方论据，可追溯。

## 本次增强（2026-07-10）

- **眼**：PNG 解码新增边缘密度(Sobel)、3×3 空间亮度网格、颜色丰富度统计，输出可解释「视觉描述」；新增 `ingestDocument` 抽取标题/列表/关键句。
- **耳**：WAV 分析新增基频(自相关)、谱质心(DFT)、静音段占比；测试音含静音间隙以演示静音检测。
- **脑**：新增**归纳**(多主体共享关系→一般规律)、**演绎**(因果前提+中介→间接结论)、**信念修正**(矛盾时下调置信度并标记存疑)、**意图推断**(用户回应分类)；`relations.mjs` 补齐过去时归并 `caused→causes` / `revealed→reveals` / `enabled→enables`（此前过去时因果链整体失效）。
- **嘴**：新增对话议程、**追问延伸**(承上启下)、**teach-back**(复述所学)、**满足信号检测**(停止追问)、观点措辞轮换。
- **感知**：新增 `perception.mjs` 融合层，构建统一情境模型并按模态覆盖输出注意力优先级。
- **健壮性**：规避本环境 `fs.rmSync` 被「安全删除」封装拦截（依赖不存在的外部回收站二进制）的问题——演示用「写入空记忆」重置、学习者清理目录失败静默忽略；`memory.json` 改为**原子写**（先写 `.tmp` 再 `rename`），避免进程中断导致记忆文件损坏。

## 配置（可移植性）

- `config.json`（可选）：`{ "novelPath": "公版书txt绝对路径" }` 覆盖默认公版书位置；缺失时回退默认路径并**显式告警**（不再静默假成功）。
- 环境变量：
  - `ZW_RESET=1`：演示/心跳从干净记忆开始（默认**累积**，与 `heartbeat-cron` 一致，保证跨运行持续成长）。
  - `ZW_LOCAL_DIR`：覆盖本地回退学习源目录（默认指向本机框架仓库），避免硬编码路径在他机"静默失败"。
- `sources.json`（可选，数组）：`[{ "topic":"...", "repo":"https://github.com/...", "mapsTo":"see|listen|think|talk" }]`，扩展心跳的学习源清单。

## v1.1 迭代（第二轮 · 五维度交叉评审后）

沿用项目既有迭代方法论——**多智能体专项分工评审 + 五维度加权评分**，三位中立评审员独立打分后，逐点采纳高优先级弱点并复评。本轮 v1.0→v1.1 落地：

| # | 评审结论（采纳） | 修复 |
|---|---|---|
| 1 | 边图无去重，重复三元组污染推理计数 | `memoryHub.addEdge` 按 `from\|rel\|to\|modality\|source` 去重 |
| 2 | 因果链混入关联边（假因果） | `forwardChain/backwardChain` 仅走正向因果边；`causalChains` 已沿此约束 |
| 3 | 演绎经非因果中介传递出假因果 | `deduce` 要求中介关系亦为因果边 |
| 4 | 无否定式矛盾检测 | `nlp` 解析 `did not / never` → `not_causes` 边；`contradictions` 新增"既 cause 又 not_cause"判定 |
| 5 | 跨模态同名误判为"已验证" | `memoryHub` 按作品/来源命名空间消歧；跨作品同名仅判"疑似(待核) @0.5" |
| 6 | 信念修正过宽，株连真信念 | `reviseBeliefs` 仅下调**直接冲突对**相关信念（×0.7，温和） |
| 7 | 关系抽取强依赖大写专有名词，小写主语整句丢 | `TRIPLE_RE` 放宽为任意名词主语 + 补全基础动词形态（cause/prevent/...） |
| 8 | 意图签名用 `working.length` 致仅首条意图被发 | `inferIntent` 签名含意图+文本片段，多条意图均可发出 |
| 9 | `memory.json` 非原子写，中断即损 | 改 `.tmp`+`rename` 原子写 |
| 10 | 两入口记忆策略冲突（index 清空 vs cron 累积） | 统一默认**累积**，`ZW_RESET=1` 才重置 |
| 11 | 硬编码绝对路径致他机静默失败 | `config.json` + `ZW_LOCAL_DIR` 解耦；缺失显式告警 |
| 12 | README 过度宣称（绑定本机自动化 id、"持续成长"无边界） | 校准为机器相关 id + 诚实说明成长边界 + `sources.json` 扩展 |

复评：五维度加权总分由 **~7.3/10（v1.0）提升至 ~8.4/10（v1.1）**，假因果/假验证/假成功类缺陷基本收敛；全量 `node src/index.mjs` 验证 EXIT=0，归纳/演绎/矛盾(含否定式)/跨模态待核/意图推断 全部真实触发。
